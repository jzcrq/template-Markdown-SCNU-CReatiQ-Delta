* [PDF 下载](https://github.com/jzcrq/template-Markdown-SCNU-CReatiQ/raw/gh-pages/template.pdf)

感觉大部分魔改自 jiangly 与 SSerxhs 的模板。

模板库中的代码均在 `C++17` 语言标准下测试，在更低的版本编译运行可能会出错。
